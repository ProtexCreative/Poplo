# Poplo
Campus connect web app.

## Run Web App
    # Install all the packages
    E:/Poplo> npm install
    E:/Poplo> cd client
    E:/Poplo/client> npm install

    # Run the server and client
    E:/Poplo> npm run dev

## Packages Used
